Security Policy
🔒 Supported Versions

Mass Control is a live web project. Security updates are applied directly to the latest version hosted on the official website:

👉 https://pixlect-studio.github.io/MASS-CONTROL/

Only the current live version is supported. Older or unofficial copies are not maintained and may be insecure.

🚨 Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

How to report:

Send details via your preferred contact method (add Discord or email below)
Include a clear description of the issue
Provide steps to reproduce the vulnerability
(Optional) Include screenshots or proof-of-concept

⚠️ Guidelines
Do NOT publicly disclose the vulnerability before it is fixed
Do NOT exploit the vulnerability beyond what is necessary to demonstrate it
Do NOT attempt to access or modify data that does not belong to you
🛡️ Scope

This policy applies only to the official version of Mass Control hosted at:

👉 https://pixlect-studio.github.io/MASS-CONTROL/

❌ Out of Scope
Modified or unofficial copies of the project
Issues caused by browser extensions or external tools
General suggestions or feature requests (use other channels for those)
